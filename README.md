## Development
```
# Clone the repository
git@github.com:chingu-voyage4/Geckos-Team-17.git

# Install required dependencies & devDependencies
yarn install

# Run the client
yarn dev:client

# Run the server
yarn dev:server

# Runt the client and server with concurrently
yarn dev

```
